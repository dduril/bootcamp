## HTML5 and CSS3

#### Lists and Links

**Unordered lists**

- `ul`
- `list-style-type` properties
	- `disc`
	- `circle`
	- `square`
	- `none`
- `li`


**Ordered lists**

- `ol`
- `list-style-type` properties
	- `decimal`
	- `decimal-leading-zero`
	- `lower-alpha`
	- `upper-alpha`
	- `lower-roman`
	- `upper-roman`
- `li`

**Description lists**

- `dl`
- `dt`
- `dd`

**`<a>` element**

- `href`
- `title`
- `tabindex`
- `accesskey`

**CSS pseudo-classes to format links**

- `link`
- `visited`
- `hover`
- `focus`
- `active`

**Removing underlines and borders**

- `text-decoration`
- `border-style`
