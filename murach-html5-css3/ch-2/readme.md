## HTML5 and CSS3

#### Coding and testing web pages

**CSS example**

- `h1` - selector
- `color` - property
- `blue` - value
- `color: blue;` - declaration

**CSS rule set:**

	h1 {
		color: blue;
	}

**Selecting by element type, id, and class**

**Type**

	body {
		font-family: Arial, sans-serif;
		font-size: 100%;
		width: 90%;
    	margin: 0 auto;
    	padding: 1em;
	}

**ID**

	#main_div {
		font-size: .9em;
		text-align: left;
	}

**Class**

	.bordered_nav {
		border: 1px solid #999;
	}
