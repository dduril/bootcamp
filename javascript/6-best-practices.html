JavaScript Best Practices

Variables, Functions, and Methods - CamelCase

var id;
var firstName;
var minimumPaymentAmountDue;
function displayName() { ... }

Objects - Uppercase First Letter

var myDate = new Date();

Brace Style

if (condition) {
    // ...
}

Always Use Blocks

if (x > 500) {
    console.log("Alert.");
}

Define functions before using them

function createMessage(firstName, lastName) {
    console.log("Greetings! " + firstName + " " + lastName); 
}

createMessage("John", "Smith");

Always include semicolon to end a statement

var myDate = new Date(2016, 8, 19, 0, 0, 0);
console.log(myDate);

Always use var when declaring a variable

var myArray = new Array();
var foo;
var bar;
